## how to use pockit?
1. install ubuntu 16.04
2. install nvidia-docker2 
3. sudo docker load -i pockit.tar 
4. sudo docker run --runtime=nvidia --name nvdk -p 1612:8888 -u root -it pockit:v1 /bin/bash
